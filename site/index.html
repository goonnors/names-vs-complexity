Результаты

Было проанализировано 224139 java файлов. Для каждого файла были получены значения Cyclomatic Complexity (CC) и количество переменных, имеющих составные названия. Значения CC оказались распределены не равномерно. Половина всех данных (49%) соответствуют значению CC=1. На рис. 1 показано распределение концентрации CC для нескольких областей в процентном соотношении.

Анализ значений при CC = 1

Разброс уникальных значений составных названий переменных при CC = 1 велик (рис. 2). Но в процентном соотношении таких файлов мало (рис. 3). В основном такие файлы вообще не имеют составных названий переменных (60% файлов).

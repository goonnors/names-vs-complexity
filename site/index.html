Результаты

Было проанализировано 224139 java файлов. Для каждого файла были получены значения Cyclomatic Complexity (CC) и количество переменных, имеющих составные названия. Значения CC оказались распределены неравномерно. Почти половина всех данных (49%) соответствуют значению CC=1. На рис. 1 показано распределение концентрации CC для нескольких областей в процентном соотношении.

Выводы

Большинство файлов имеет минимальную сложность CC = 1, но они интереса для этого исследования не представляют. Для случаев когда программы имеют среднюю сложность (CC ~ 5), большую сложность (CC ~ 10) и очень большую сложность (CC ~ 20) явной зависимости CC от compound vars нет. Зависимость начинает прослеживаться при экстремально больших значениях CC, но такие ситуации очень редки.

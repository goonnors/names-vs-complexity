Результаты

Было проанализировано 224139 java файлов. Для каждого файла были получены значения Cyclomatic Complexity (CC) и количество переменных, имеющих составные названия. Значения CC оказались распределены неравномерно. Почти половина всех данных (49%) соответствуют значению CC=1. На рис. 1 показано распределение концентрации CC для нескольких областей в процентном соотношении.

Анализ значений compound names при CC = 1

На рис. 3 показано распределение составных названий переменных при CC = 1. 60% данных не имеют составных названий переменных. 16% имеют 1 переменную, ещё 16% имеют небольшую сложность CC < 5, и 8% - остальные файлы.

На рис. 2 показаны уникальные значения составных названий переменных.
